# Conference Center Website

A responsive multi-page conference center website built using HTML5 and CSS3.  
This project focuses on semantic HTML structure, clean layout design, and reusable styling across multiple pages.

## Project Overview

The Conference Center website provides visitors with information about the venue, hosted events, expert speakers, testimonials, and a contact form for inquiries or bookings. The site is designed with a professional, modern aesthetic appropriate for corporate, educational, and industry events.

## Technologies Used

- HTML5 (semantic markup)
- CSS3 (layout and responsive styling)
- Git and GitHub for version control

## Pages Included

- Home (`index.html`) – Landing page introducing the conference center
- About (`about.html`) – Overview of the venue, mission, and facilities
- Events (`events.html`) – Event categories and an event schedule table
- Contact (`contact.html`) – Contact form for general inquiries and bookings

## Features

- Semantic HTML elements including `header`, `nav`, `main`, `section`, `article`, and `footer`
- Consistent navigation across all pages
- Responsive layout using a single shared stylesheet
- Event listings with images and descriptive content
- Event schedule table
- Images with descriptive alt text for accessibility
- Contact form with text, email, and message inputs
